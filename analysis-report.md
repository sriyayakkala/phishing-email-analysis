# Phishing Email Investigation Report

# Incident Summary
A suspicious email claiming to be from PayPal was analyzed for phishing indicators.

# Key Findings
- Spoofed sender domain detected  
- Malicious link directing to fake website  
- Use of urgency and fear tactics  

# Risk Assessment
High risk of credential theft if user clicks the link.

# Indicators of Compromise (IOCs)
- Fake domain: paypaI.com  
- Malicious URL: paypal-verification-secure.com
  
# Recommended Actions
- Block malicious domain  
- Educate users on phishing awareness  
- Monitor for similar emails  

## 📚 Conclusion
The email is a clear phishing attempt designed to steal user credentials.
